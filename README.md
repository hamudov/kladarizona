# Скрипт управления кладами для Arizona RP

## 📍 Описание

Это Lua скрипт для SA:MP Arizona RP, который позволяет игрокам добавлять и управлять метками кладов на карте с использованием графического интерфейса прямо в игре.

## 🎮 Основные возможности

- ✅ **Графический интерфейс** через команду `/klua`
- ✅ **Добавление точек** с текущей позиции игрока
- ✅ **Добавление по координатам** - вручную указать X, Y, Z
- ✅ **Выбор на карте** - двойной клик на карте для выбора точки
- ✅ **Отмечание на карте** - красные метки для каждой точки
- ✅ **Список всех точек** с расстоянием до каждой
- ✅ **Удаление точек** - удалить отдельную точку или все
- ✅ **Сохранение/загрузка** - сохранение в JSON файл
- ✅ **Информация о координатах** - команда `/klua coords`

## 📋 Команды

### Главная команда
```
/klua
```
Открывает/закрывает главное окно управления

### Подкоманды
- `/klua add [название]` - Добавить точку с текущей позиции
  - Пример: `/klua add Клад у дерева`

- `/klua del [ID]` - Удалить точку по ID
  - Пример: `/klua del 1`

- `/klua list` - Показать список всех точек

- `/klua coords` - Показать текущие координаты игрока

- `/klua clear` - Удалить все точки (осторожно!)

- `/klua help` - Показать справку

## 🖱️ Использование графического интерфейса

### Главное меню
```
[ 1 ] Добавить точку           - Добавляет точку из текущей позиции
[ 2 ] Добавить по координатам  - Ввести координаты вручную (X, Y, Z)
[ 3 ] Удалить точку            - Выбрать и удалить точку
[ 4 ] На карте                  - Выбрать позицию двумя кликами на карте
[ 5 ] Сохранить                - Сохранить все точки в файл
[ ESC ] Закрыть                 - Закрыть окно
```

### Режим "Добавить точку"
1. Встаньте на нужное место в игре
2. Откройте `/klua`
3. Нажмите `[ 1 ]`
4. Введите название точки (например: "Клад у пустыни")
5. Нажмите `ENTER` - точка добавлена и отмечена на карте!

### Режим "Добавить по координатам"
1. Откройте `/klua`
2. Нажмите `[ 2 ]`
3. Используйте цифры для ввода координат X, Y, Z
4. Нажимайте `TAB` для переключения между полями
5. Введите название
6. Нажмите `ENTER`

### Режим "На карте" (двойной клик)
1. Откройте `/klua`
2. Нажмите `[ 4 ]`
3. **Нажмите ЛКМ на карту** - первая точка (система запомнит)
4. **Нажмите ЛКМ на карту второй раз** - вторая точка подтверждает выбор
5. Введите название
6. Нажмите `ENTER` - готово!

## ⌨️ Быстрые клавиши в интерфейсе

| Клавиша | Действие |
|---------|----------|
| `1-5` | Быстрый выбор опций в главном меню |
| `TAB` | Переключение между полями ввода |
| `ENTER` | Подтверждение действия |
| `ESC` | Отмена/закрытие окна |
| `BACKSPACE` | Удалить символ |
| `0-9` | Ввод цифр для координат |
| `Буквы` | Ввод названия точки |

## 🎨 Визуальные элементы

- 🔴 **Красные точки** - отмечают локации кладов на карте
- 🟡 **Желтый текст** - текущие координаты
- 🟢 **Зелёный текст** - успешные действия
- 🔴 **Красный текст** - важные предупреждения/ошибки

## 🔧 Установка

1. Загрузите файлы в папку скриптов вашего клиента
2. Убедитесь, что скрипт загружен при запуске клиента
3. Используйте команду `/klua` для открытия интерфейса

## 📝 Примеры использования

### Пример 1: Добавить точку из текущей позиции
```
/klua
[нажми 1]
Введи: Древний храм
[ENTER]
```

### Пример 2: Добавить по точным координатам
```
/klua
[нажми 2]
X: 1234 [TAB]
Y: -567 [TAB]

Название: Пещера с кладом
[ENTER]
```

```
/klua
[нажми 4]
[ЛКМ на карту - первая точка]
[ЛКМ на карту - вторая точка]
Название: Загадочная локация
[ENTER]
```

## 🐛 Возможные проблемы

**Q: Точка не отмечается на карте**
A: Убедитесь, что скрипт правильно загружен и интегрирован с SA:MP API

**Q: Координаты не сохраняются**
A: Проверьте прав доступа на запись в папку скриптов

**Q: Окно не открывается**
A: Убедитесь, что используется корректная команда `/klua` (без доп. параметров для открытия)
